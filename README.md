# Dog
### A simple program that prints a file content on the screen.
Thats a program built as a "copy" of the cat linux program for learning porpouses. 

## Usage
1. Use `./dog <your_file>` to run the program or `./dog <your_file> -n` to show the lines.
2. You can use dog without args to write in the terminal and redirect the stdout to a file.
